To run Monte Carlo algorithm to calculate PI
All commands run on macOS.

For C++ Code
Compile:
g++ -O3 -std=c++23 -o MonteCarloPI.exe MonteCarloPI.cpp
("g++ -O3 -std=c++17 -S MonteCarloPI.cpp" produces assembly code in MonteCarloPI.s)
Run:
MonteCarloPI.exe

For Python
Run
python3 MonteCarloPI.py

For other platforms you can use whatever compiler and interpreter for C++ and Python you want.

For Java
Compile:
javac MonteCarloPI.java
Run:
java MonteCarloPI
